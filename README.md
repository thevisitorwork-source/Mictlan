# Mictlan
Mi juego del Mictlán
Características principales:

Tema del Mictlán con una estética muy bonita y atmosférica

Sistema de nombres permitidos (Victoria, Emilia, Rosaura, Eugenia, Ignacio, Arturo, Ness)

Juego de adivinanza de números entre 1 y 100

Poemas personalizados para cada personaje (calaveritas literarias)

Sistema de progreso desbloqueando nombres gradualmente

Efectos visuales con pétalos cayendo

Música de fondo ("La Llorona")

Panel administrativo con tracking de jugadores

Generación de PDF con las crónicas completas

Mecánicas únicas:

Validación de nombres con formato específico

Sistema de intentos con barra visual

Última oportunidad con adivinanza del 1-5

Poemas que se revelan según el progreso

Diseño responsive que implementamos:

Adaptable a móviles, tablets y escritorio

Pétalos optimizados para cada dispositivo

Botones touch-friendly

Layouts flexibles
